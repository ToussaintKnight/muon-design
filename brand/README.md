## 品牌核心

**精神母题**：Everything that rises must converge. 凡向上生长者，终将汇合。

**主标语**：人类赋予方向，智能折叠时间。

**命名来源**：MUON（缪子）是一种轻子，质量约为电子的 207 倍，由卡尔·安德森于 1936 年在云室实验中发现。

## 颜色系统（基于真实物理实验）

| 名称 | 色值 | 物理来源 |
|------|------|---------|
| 契伦科夫蓝 | `#0077BE` | 带电粒子超光速时发出的契伦科夫辐射 |
| 电离白 | `#E8E8E0` | 云室/气泡室中粒子轨迹的白色雾迹 |
| 轨迹金 | `#C4A84A` | 高能粒子在量能器中的能量沉积可视化 |
| 闪烁体紫 | `#6B5B95` | 闪烁体探测器中的光子发射 |
| 磁体银 | `#A8B4C0` | 加速器超导磁体与束流管的金属表面 |
| 暗室黑 | `#08080C` | 粒子探测的暗室环境 |
