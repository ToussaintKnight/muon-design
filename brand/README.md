## 设计原则

1. **向上生长者终将汇合** — 所有设计元素都暗示汇聚、连接、成长
2. **人类赋予方向，智能折叠时间** — 人类元素与智能元素并存
3. **拥有交点，不拥有关系** — 设计不过度侵入，留白和边界是设计的一部分
4. **默认安全，逐步授权** — 视觉层次清晰区分"Agent 建议"与"人类决策"
5. **复古未来，中文可读** — 美学大胆，但绝不牺牲中文阅读体验
